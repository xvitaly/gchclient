Garant Checker Offline (служба проверки пользователей) — клиент, демонстрирующий работу с API проекта PHP Garant Checker. Программа написана исключительно for fun для гарантов и пользователей форума team-fortress.su.

Данное приложение предназначено для гарантов и пользователей, которые вынуждены проверять десятки или даже сотни пользовательских профилей Steam на наличие в чёрном/белом списках, гарантах и т.д.

При закрытии окна она будет свёрнута в системный трей и продолжит работу. Для окончательного завершения работы и выгрузки из памяти нажмите правой кнопкой по значку в системном трее (около часов Windows) и выберите пункт Выход.

Свёрнутая программа автоматически мониторит буфер обмена на наличие в нём ссылок на профили Steam и при обнаружении автоматически проверяет их в фоновом режиме. По окончании проверки появится окно, показанное на скриншоте, с её результатами.

В окне результатов проверки Вы можете скопировать SteamID пользователя в буфер обмена, просто щёлкнув по нему либо по нику левой кнопкой мыши.

Постоянную ссылку на профиль можно либо скопировать в буфер (нажмите правой кнопкой и выберите нужный вариант), либо сразу перейти по ней (щелчок левой кнопкой).
